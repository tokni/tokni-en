# tokni.com (English)

Static HTML/CSS website for Tøkni, an IT consultancy based in the Faroe Islands.

Served at [tokni.com](https://tokni.com) via GitHub Pages.

## Structure

- `index.html` - Landing page
- `css/main.css` - Styles
- `privacy_policy.html` - Privacy policy

## Development

Serve locally with any static file server:

```bash
npx serve
```
